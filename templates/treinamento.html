Como treinar seu modelo de detecção de objetos

Siga os passos abaixo para criar um dataset e treinar um modelo personalizado:

  1. Carregue imagens que contenham objetos que você deseja detectar
  2. Desenhe retângulos em torno de cada objeto na imagem
  3. Selecione a classe correta para cada retângulo
  4. Salve as anotações
  5. Repita o processo com pelo menos 10 imagens diferentes
  6. Inicie o treinamento quando tiver anotações suficientes

Dica: Quanto mais imagens você anotar, melhor será o modelo. Recomendamos pelo menos 50 imagens com boa diversidade de situações.

Ajuda e Dicas
Como anotar imagens:
  1. Carregue uma imagem usando o botão acima
  2. Clique e arraste na imagem para desenhar um retângulo
  3. Selecione a classe do objeto no menu suspenso
  4. Clique em "Salvar Anotação" para preservar as marcações
Dicas para melhores resultados:
  • Use imagens de alta qualidade e boa iluminação
  • Inclua objetos em diferentes ângulos e distâncias
  • Varie os cenários de fundo para melhor generalização
  • Seja consistente nas anotações entre imagens semelhantes
